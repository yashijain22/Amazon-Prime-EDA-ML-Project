**#Amazon-Prime-EDA-ML-Project**

This project performs Exploratory Data Analysis (EDA) and builds a machine learning model on the Amazon Prime TV Shows and Movies dataset.

## Dataset
Two datasets are used:
- titles.csv
- credits.csv

## Project Tasks
- Data exploration
- Data cleaning
- Dataset merging
- Data visualization
- Machine learning model to predict IMDb scores

## Technologies Used
Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Scikit-learn
